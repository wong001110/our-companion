# Core Terminology

## Signal

A raw observation from the world.

Examples:

- Hacker News item
- GitHub repo
- RSS post
- user action
- local file update
- future Companion visit message

## Experience

Something Ann or the user has gone through.

Examples:

- Ann browsed the web
- Ann visited another Companion
- user discussed an idea
- user completed a task
- user ignored several discoveries

## Discovery

A signal that may be worth attention.

## Concept

A normalized idea or topic extracted from discoveries.

Example:

```txt
SQLite-backed personal memory tools
```

## Insight

A meaningful interpretation of concepts.

Example:

```txt
Local-first architecture may be important for private personal AI memory.
```

## Knowledge

A stable understanding that should influence future reasoning.

## Journey

A long-running exploration path.

## Curiosity Gap

Something the user likely needs to learn next.

## Decision

The system's determination of whether Ann should speak, wait, remember, ignore, or act.

## Expression

The visible or audible output: bubble, card, animation, notification, voice.

## Performance

The acted-out animation sequence that makes Ann feel like she is doing something.

Example:

```txt
run_to_chrome -> tap -> type -> confirm -> return
```

## Command

The real system-level action.

Example:

```txt
openUrl("https://youtube.com")
```
