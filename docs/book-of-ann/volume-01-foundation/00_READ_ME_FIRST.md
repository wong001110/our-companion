# Book of Ann — Volume 01
# Foundation: Evolution Edition

This volume is the foundation pack for evolving an existing Our Companion project into the long-term Companion Cognitive Architecture.

This is not a greenfield PRD.
It assumes the project already exists and must be refactored safely.

## How Argent Should Use This Volume

1. Read `01_ARGENT_MASTER.md` first.
2. Read `02_PROJECT_EVOLUTION_CONTEXT.md`.
3. Follow the migration sequence in `12_MIGRATION_STRATEGY.md`.
4. Do not rewrite the entire project.
5. Preserve working behavior while introducing the new architecture step by step.
6. Prefer adapters, events, interfaces, and compatibility layers before destructive refactors.

## Volume 01 Scope

This volume defines:

- Product vision
- Companion principles
- Cognitive loop
- Target architecture
- Existing-project migration strategy
- Event-driven foundation
- Local-first and future-cloud rules
- Coding and documentation standards
- The task execution model for future volumes

## Out of Scope

This volume does not fully implement:

- Discovery Engine internals
- Curiosity Engine scoring
- Character Creator UI
- Companion Society networking
- Asset upload pipeline
- Voice or TTS

Those belong to later volumes.
