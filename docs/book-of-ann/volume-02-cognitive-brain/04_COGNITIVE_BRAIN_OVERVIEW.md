# Cognitive Brain Overview

The cognitive brain decides what something means before Ann decides how to express it.

## Full Cognitive Flow

```txt
SignalCaptured
  ↓
SignalNormalized
  ↓
DiscoveryCandidateCreated
  ↓
DuplicateCheckCompleted
  ↓
DiscoveryCreated
  ↓
ConceptMatched or ConceptCreated
  ↓
PatternDetected
  ↓
InsightGenerated
  ↓
CuriosityGapMatched
  ↓
DecisionRequested
  ↓
CompanionDecisionMade
```

## Core Separation

### Thinking

Answers:

```txt
What is this?
What does it relate to?
Is it new?
What does it mean?
```

### Curiosity

Answers:

```txt
Does this help the user grow?
Does this fill a gap?
Is this part of the current season?
Is this worth spending curiosity budget on?
```

### Decision

Answers:

```txt
Should Ann act on this?
Should she speak now?
Should she save it silently?
Should she wait?
```

## Why This Separation Matters

Without this separation, the system becomes a recommendation feed.

With this separation, Ann becomes a companion who understands, filters, and respects timing.
