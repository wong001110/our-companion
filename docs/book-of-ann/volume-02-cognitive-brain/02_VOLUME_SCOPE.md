# Volume 02 Scope

## Included

This volume defines:

- Signal Engine
- Experience abstraction
- Thinking Engine
- Discovery lifecycle
- Deduplication
- Concept model
- Pattern detection
- Insight generation
- Curiosity Engine
- Curiosity Budget
- Curiosity Gap
- Curiosity Debt
- Curiosity Season
- Novelty and relevance scoring
- Attention model
- Decision Engine
- Cognitive event flow
- LLM usage boundaries
- Migration from current engine packages

## Excluded

This volume does not define:

- final speech bubble UI
- full animation state machine
- creator asset upload
- voice selection
- cloud sync implementation
- society relationship system
- database schema migrations in detail

## Output Expectations

After this volume, Argent should be able to build the cognitive backend foundation while keeping the existing desktop companion experience functional.
