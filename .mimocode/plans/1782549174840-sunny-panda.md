# Plan: Fix Discovery Announcement UX — Separate Ann Speech from Discovery Card

## Problem

When a discovery is announced, Ann speaks the `short_message` but no visual card appears. The task requires: Ann says only a short sentence, and a pop-out discovery card shows the details separately.

## Current Data Flow

```
DiscoveryShareOrchestrator.announceDiscovery()
  → generateReason(discovery) → DiscoveryReason { why_this_matters, recommended_action, short_message, tags }
  → emits AnnMessageQueued with { discoveryId, title, message: reason.short_message }
  → ElectronIpcBroadcaster forwards 'discovery:announce' to renderer
  → CompanionShell.onAnnounce → showTypewriterSpeech(payload.message)
```
