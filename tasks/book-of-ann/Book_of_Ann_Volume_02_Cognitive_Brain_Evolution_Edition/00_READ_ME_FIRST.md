# Book of Ann — Volume 02
# Cognitive Brain: Evolution Edition

This volume defines the core cognitive system of Our Companion.

It builds on Volume 01 Foundation and assumes an existing project is being evolved, not rewritten.

## Read Order

1. `01_ARGENT_MASTER.md`
2. `02_VOLUME_SCOPE.md`
3. `03_CURRENT_TO_TARGET_COGNITIVE_MAPPING.md`
4. `04_COGNITIVE_BRAIN_OVERVIEW.md`
5. Then follow the engine chapters in order.

## Volume 02 Goal

Create the cognitive foundation that allows Ann to:

- observe signals
- understand discoveries
- identify concepts
- detect patterns
- generate insights
- manage curiosity
- protect attention
- decide what deserves action
- prepare information for expression

## Out of Scope

This volume does not implement:

- detailed character animation
- creator UI
- asset upload
- cloud sync
- Companion Society networking
- full desktop command execution

Those are covered in later volumes.
