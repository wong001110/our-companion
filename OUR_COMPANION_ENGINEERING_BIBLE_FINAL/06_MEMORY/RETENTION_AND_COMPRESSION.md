# Memory Retention and Compression

## Strategy
Important memories are retained.
Low-importance raw items are compressed into summaries.

## Importance signals
- added to journey
- referenced multiple times
- linked to outcomes
- accepted by user
- related to active interests
- related to character expertise

## Compression
Old low-importance discoveries become:
- summary
- tags
- relation edges
- source URL if available

Never delete user-created notes without user action.
