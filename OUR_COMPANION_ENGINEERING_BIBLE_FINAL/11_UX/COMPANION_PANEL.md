# Companion Panel UX

## Tabs
1. Home / Character Room
2. Discovery
3. Journey
4. Memory
5. Ask

## Home / Character Room
Shows:
- active character
- current thought/intent
- current journey
- recent discovery
- diary highlight
- active companions list in future

## Discovery
Exploration board.
Cards include:
- title
- source
- why character thinks it matters
- actions: view, later, not interested, add to journey

## Journey
Adventure timeline.

## Memory
Character notebook, editable memory.

## Ask
Text input, task execution, discussion.
