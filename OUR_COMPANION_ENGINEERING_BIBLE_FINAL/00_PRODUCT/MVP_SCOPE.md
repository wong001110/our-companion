# MVP Scope

## Must build
- Electron desktop app.
- Transparent always-on-top companion window.
- React companion panel.
- PixiJS character renderer.
- Text input command bar / Ask panel.
- DeepSeek V4 Flash integration.
- SQLite local-first memory.
- Character Runtime: state + emotion + intent + behavior loop.
- Discovery Engine: GitHub, Reddit, Hacker News, YouTube.
- Discovery ranking and daily cap.
- Tool Engine:
  - open_url
  - open_app
  - search_web
  - basic browser navigation
- Companion Panel:
  - Home / Character Room
  - Discovery
  - Journey
  - Memory
  - Ask
- Journey Timeline.
- Memory Graph tables.
- Character Diary.
- User-editable memory.

## Explicitly not in MVP
- Voice / STT / TTS.
- Login.
- Cloud sync implementation.
- Marketplace.
- External character import UI.
- Live2D.
- Mobile app.
- Payment actions.
- Login automation.
- Sending messages on behalf of the user.
- Deleting user data without explicit user action.
