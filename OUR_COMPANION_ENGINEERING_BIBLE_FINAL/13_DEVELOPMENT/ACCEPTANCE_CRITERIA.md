# Acceptance Criteria

## MVP is acceptable when:
- App launches.
- Desktop companion appears.
- Companion can idle/walk/sleep/think/discover/execute.
- Clicking companion opens panel.
- User can input text command.
- DeepSeek can classify tool intent.
- open_url/search_web/open_app work.
- Discovery feed can fetch at least 2 sources.
- Discoveries are ranked and capped.
- User can save discovery to journey.
- Memory node and edge are created.
- Journey timeline displays saved exploration.
- Diary entry can be generated.
- Memory can be edited/deleted.
- No blocked high-risk action can execute.
