# Discovery Engine

## Sources
- GitHub
- Reddit
- Hacker News
- YouTube

## Pipeline
```text
fetch source data
 -> normalize
 -> deduplicate
 -> score
 -> AI summarize
 -> character filter
 -> store candidate
 -> share under daily cap
```

## Daily cap
Hard cap: 10 shared discoveries/day globally.
Normal expected range: 1-5/day.
Character personality, relationship, and user activity affect actual frequency.

## Status lifecycle
candidate -> shared -> saved/rejected/ignored -> memory/journey optional.
